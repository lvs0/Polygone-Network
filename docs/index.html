// capabilities

Built for threat models
where the relay is untrusted.

🔒

Relay Blind

Messages are end-to-end encrypted before they leave your device. The relay only ever sees encrypted blobs — it cannot read, inspect, or log message content.

⚡

Invisible Routing

Traffic analysis resistance built in. Packet sizes and timing are normalised across the network, making it computationally hard to correlate sender and receiver.

🌐

Distributed Mesh

No single point of failure. Nodes self-organise into a distributed mesh topology, routing around censorship and network partitions automatically.

// protocol

How it works

┌─────────────────────────────────────────────────────────────────┐
│                         POLYGONE NETWORK                         │
└─────────────────────────────────────────────────────────────────┘

        ALICE                              BOB
     [device: encrypted]            [device: encrypted]
           │                                          ▲
           │  ╔════════════════════╗                  │
           │  ║  relay sees: garbage  ║
           │  ║  ←── encrypted ──→ ║                  │
           │  ╚════════════════════╝                  │
           ▼           RELAY              ─────────────

  1. Alice encrypts with Bob's public key                     
  2. Encrypted payload sent through relay — relay cannot decrypt
  3. Bob's device holds the only private key                   

  relay sees nothing · zero knowledge · e2e everywhere

Payloads are encrypted at the application layer. The relay is a dumb pipe — it forwards bytes it cannot interpret.

// security model

Defence in depth.
Four layers of protection.

1

App-layer E2E encryption

Curve25519 key exchange + ChaCha20-Poly1305 AEAD. Every message is sealed before it touches the network.

2

Relay-blind routing

The relay receives ciphertext only. No metadata leakage — source, destination, and content are all protected.

3

Traffic analysis resistance

Constant-rate padding and randomised timing. Adversaries watching the wire see indistinguishable patterns.

4

No logs, no identity

Polygone nodes can be configured with zero persistence. Boot with a RAM-only disk image — nothing to subpoena.
