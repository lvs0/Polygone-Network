# Security Policy

## Supported Versions

| Version | Supported          |
|---------|-------------------|
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:               |

## Reporting a Vulnerability

Please report security vulnerabilities by emailing **polygone@proton.me** (Lévy) or through the GitHub Security Advisories tab.

We appreciate your help in responsibly disclosing security issues. Please include:
- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Possible mitigations (if known)

We will acknowledge receipt of your report within 48 hours and provide a preliminary assessment within 7 days.

## Preferred Languages

We prefer all communications to be in French or English.
