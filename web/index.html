How it works
Messages exist only in transit
Every message is encrypted post-quantum, split into 7 fragments, and routed through nodes that see nothing — not the content, not the sender, not the receiver.

The stack
Post-quantum by default
Every layer uses standard, peer-reviewed algorithms. No Snake Oil. No marketing. Real math.

NIST FIPS 203
Post-quantum key exchange. Quantum computers cannot break it in reasonable time.
In production
ML-DSA-87
NIST FIPS 204
Post-quantum digital signatures. Identity and non-repudiation.
In production
AES-256-GCM

Status
Status by feature
Not just a roadmap. A realistic overview of what is currently implemented, in progress, or planned.
💬
Ephemeral Messaging Building
Messages encrypted end-to-end, fragmented through the mesh, TTL-based expiry. No server, no logs, no metadata.
🔐
Post-Quantum Crypto Live
ML-KEM-1024 key exchange + ML-DSA-87 signatures. Quantum computers cannot break the encryption.
🖥️
Compute Lending Planned
Idle GPU/CPU cycles lent to peers. Earn karma. Run distributed LLM inference via the Petals protocol.
📦
Distributed Drive Planned
Files encrypted, sharded 4-of-7, distributed across peers. Recover from any 4 of 7. No single point of failure.
🌐
Hide Proxy Planned
SOCKS5 proxy through the mesh. No exit node logs. No ISP tracking. Traffic looks like random encrypted bytes.
🧠
AI Orchestrator Planned
Plan-based AI orchestration. Local LLM interface (Ollama, llama.cpp, Notch SLM). SSE events. Extensible service model.
🔀
P2P Transport Building
libp2p with Kad DHT, GossipSub, Relay, mDNS, AutoNAT, DCUtR. NAT traversal built-in. Zero port forwarding required.
⏱️
TUI Dashboard Building
ratatui v2. Five tabs. Sparklines. Real-time metrics. Arrow-key navigation. Zero configuration.
Get started
Build from source
Currently developer-only. Requires Rust 1.75+. Pre-compiled binaries for regular users coming soon.
$ bash — clone + build
$ git clone https://github.com/lvs0/Polygone-Network.git
# Clones into Polygone-Network/

$ cd Polygone-Network

$ cargo build --release
# Compiles all binaries (~2 min on modern hardware)
   Compiling polygone v1.0.0
    Finished release [optimized] target(s) in 1m 54s

$ ./target/release/polygone --help
Post-quantum ephemeral transit network.

  menu     Launch the interactive TUI dashboard
  status   Show system status
  test     Run the crypto self-test suite
  serve    Start the web dashboard on :8080
  send     Encrypt and fragment a message
  receive  Reconstruct a message from fragments
